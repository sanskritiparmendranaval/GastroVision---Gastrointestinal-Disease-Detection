# GastroVision---Gastrointestinal-Disease-Detection
A Compact GI Endoscopy Dataset Discover GastroVision, an open-access dataset with 8,000 GI endoscopy images across 27 classes, covering landmarks, abnormalities, polyp removals, and normal findings. Collected from Baerum Hospital and Karolinska University (Sweden), it’s annotated by expert endoscopists. Perfect for AI-driven GI disease detection
