# GastroVision - Gastrointestinal Disease Detection

## Overview

GastroVision is a deep learning-based project for detecting and classifying gastrointestinal diseases from endoscopic images. The project aims to assist healthcare professionals by automatically identifying different gastrointestinal conditions using medical image analysis.

## Features

* Automatic classification of gastrointestinal diseases from endoscopic images.
* Image preprocessing and analysis using Python.
* Jupyter Notebook implementation for training and evaluation.
* Dataset-driven approach for disease detection.
* Easy-to-understand workflow for experimentation and further research.

## Dataset

The project uses a gastrointestinal endoscopy image dataset containing images from multiple disease categories. The dataset includes various gastrointestinal conditions and normal cases for model training and evaluation.

**Note:** The complete image dataset is not included in this repository due to its large size.

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/sanskritiparmendranaval/GastroVision---Gastrointestinal-Disease-Detection.git
```

2. Navigate to the project directory:

```bash
cd GastroVision---Gastrointestinal-Disease-Detection
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Open the Jupyter Notebook:

```bash
jupyter notebook notebookf1332d64e6.ipynb
```
